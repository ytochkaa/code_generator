# Генератор кода

Учебный проект: генератор исходного кода классов на трёх языках — C++, C# и
Java. Целевой язык задаётся не на этапе компиляции, а во время выполнения —
через выбор конкретной фабрики.

## Паттерн «Абстрактная фабрика»

Генерация построена на паттерне «Абстрактная фабрика». Интерфейс `Factory`
объявляет методы создания элементов кода (`createClass`, `createMethod`,
`createPrintOperator`) и обёртки готовой программы (`wrapProgram`). Функция
`generateProgram` собирает дерево элементов, **не зная целевого языка** — он
определяется переданной фабрикой:

- `CppFactory` создаёт элементы `Cpp*` и генерирует C++;
- `CSharpFactory` создаёт элементы `CSharp*` и генерирует C#;
- `JavaFactory` создаёт элементы `Java*` и генерирует Java.

Абстрактные элементы (`Unit`, `ClassUnit`, `MethodUnit`, `PrintOperatorUnit`)
хранят общие данные, а конкретную генерацию выполняют наследники для каждого
языка.

## Модификаторы

Помимо общих для C++ модификаторов, поддержаны модификаторы, которых в C++ нет:

| Модификатор         | C++ | C#  | Java |
|---------------------|-----|-----|------|
| static              |  +  |  +  |  +   |
| const               |  +  |  -  |  -   |
| virtual             |  +  |  +  |  -   |
| abstract            |  -  |  +  |  +   |
| override            |  -  |  +  |  -   |
| sealed              |  -  |  +  |  -   |
| final               |  +  |  -  |  +   |
| internal            |  -  |  +  |  -   |
| protected internal  |  -  |  +  |  -   |
| private protected   |  -  |  +  |  -   |
| package-private     |  -  |  -  |  +   |

Каждая фабрика отображает общий набор флагов в допустимый для своего языка
синтаксис, отбрасывая неприменимые модификаторы.

## Сборка

```
cmake -B build
cmake --build build
```

Требуется компилятор с поддержкой C++17.

## Запуск

```
./build/code_generator
```

Программа печатает сгенерированный код для каждого языка в консоль и
сохраняет его в каталог `generated/` (`MyClass.cpp`, `MyClass.cs`,
`MyClass.java`).

## Проверка компиляции

Эталонные результаты генерации лежат в каталоге `tests/`. Сгенерированный код
компилируется штатными компиляторами:

```
g++ -std=c++17 -c generated/MyClass.cpp
dotnet build           # для MyClass.cs
javac generated/MyClass.java
```

Все три файла компилируются без ошибок.

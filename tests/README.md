# Тесты

Каталог содержит эталонные результаты генерации и описание проверок.

## Тестовый класс

Генератор строит абстрактный класс `MyClass`, наследник `BaseClass`, в котором
задействованы все поддерживаемые модификаторы:

- `draw` — переопределение виртуального метода базового класса (`override` +
  `sealed` в C#);
- `testFunc1` — обычный публичный метод;
- `testFunc2` — `private static final`;
- `testFunc3` — `virtual` + `const`;
- `testFunc4` — `protected static` с выводом строки;
- `testFunc5` — абстрактный метод;
- `testFunc6` — метод с доступом `internal`.

## Эталонные выводы

- `MyClass.cpp` — ожидаемый результат для C++;
- `MyClass.cs` — ожидаемый результат для C#;
- `MyClass.java` — ожидаемый результат для Java.

## Проверка компиляции

Каждый эталон компилируется штатным компилятором языка и не должен давать
ошибок:

```
g++ -std=c++17 -c MyClass.cpp
dotnet build         # проект с MyClass.cs
javac MyClass.java
```
